# ✨ Instances Animator

Procedural Geometry Nodes add-on for Blender.
Quickly create animated instance systems with randomization, looping motion, and surface scattering.

![Blender](https://img.shields.io/badge/Blender-4.2%2B-orange)
![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT--0-green)

---

## 🚀 Features

* 🎞️ Procedural instance animation
* 🎲 Random rotation & scale
* 🌊 Noise-based motion distortion
* 🔁 Looping animation system
* 🧩 Collection instancing
* 🎨 Material assignment
* ⚡ Fast Geometry Nodes workflow
* 🛠️ Non-destructive setup

---

## 📦 Installation

1. Download ZIP
2. Open Blender
3. Go to:

```text id="svy3yx"
Edit → Preferences → Add-ons → Install
```

4. Select the ZIP archive
5. Enable **Instances Animator**

---

## 🖥️ Usage

1. Select a mesh object
2. Open Sidebar (`N`)
3. Go to:

```text id="8hjff1"
Instances Animator
```

4. Press:

```text id="fwj4mg"
Apply Instances Animator
```
